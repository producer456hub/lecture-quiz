# Mollhoff Lecture Quiz — Weeks 4–6

Interactive 100-question multiple-choice quiz covering Dr. Mollhoff's lecture material on tissue types, histology, the integumentary system, wound healing, homeostasis/feedback, and the cell cycle.

## Features

- 100 MCQs, tagged by topic across 14 categories
- Sequential or shuffled order
- Optional instant feedback after each answer
- End-of-quiz results: overall score, per-topic performance bars, and a flagged "weak areas" list (any topic under 70%)
- **Retake-missed loop:** after the first round you can retake only the questions you got wrong, and keep looping until you score 100%
- Full review of every missed question with your pick vs. the correct answer

## Run it

Just open `index.html` in any modern browser — no build step, no dependencies.
